## An wen richtet sich das Nutzerhandbuch?

### Primäre Zielgruppe:

1. **Endnutzer der Software**
   - Personen, die die Software täglich nutzen
   - Verschiedene Rollen (z.B. Sachbearbeiter, Manager, Analysten)
   - Verschiedene Erfahrungsstufen

2. **Anfänger:**
   - Erste Erfahrungen mit der Software
   - Benötigen grundlegende Einführung
   - Schritt-für-Schritt-Anleitungen

3. **Fortgeschrittene Benutzer:**
   - Kennt die Grundfunktionen
   - Sucht nach erweiterten Funktionen
   - Möchte Effizienz steigern

4. **Experten:**
   - Kennt die meisten Funktionen
   - Sucht nach spezifischen Details
   - Möchte Best Practices lernen

### Sekundäre Zielgruppe:

- Neue Benutzer (Onboarding)
- Support-Personal (als Referenz)
- Trainer und Schulungsverantwortliche
